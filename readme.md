Initial step to make a node server

Type in terminal
    |npm init -y
    |npm i express chance shuffle-array
    |npm i --save-dev nodemon

Add some text in package.json
    |"scripts": {
    |    "start": "node index.js",
    |    "dev": "nodemon index.js"
    |},

Create index.js file (node server)
    |write some code

Run the server
    |npm run dev
